{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} MIMIC-IV is separated into “modules” to reflect the provenance of the data. The modules below are currently available. {{% /blocks/lead %}} {{< blocks/section color="primary-light" >}} {{% blocks/feature icon="fa-lightbulb" title="MIMIC-core" url="/iv/modules/core" %}} All patients across all modules are in MIMIC-core. {{% /blocks/feature %}} {{% blocks/feature icon="fa-lightbulb" title="MIMIC-hosp" url="/iv/modules/hosp" %}} The Hosp module provides all data acquired from the hospital wide electronic health record. {{% /blocks/feature %}} {{% blocks/feature icon="fa-lightbulb" title="MIMIC-ICU" url="/iv/modules/icu" %}} The ICU module contains information collected from the clinical information system used within the ICU. {{% /blocks/feature %}} {{% blocks/feature icon="fa-lightbulb" title="MIMIC-ED" url="/iv/modules/ed" %}} The ED module contains data for emergency department patients collected while they are in the ED. {{% /blocks/feature %}} {{% blocks/feature icon="fa-lightbulb" title="MIMIC-CXR" url="/iv/modules/cxr" %}} MIMIC-cxr provides chest x-ray data, where available, for patients from the clinical database. {{% /blocks/feature %}} {{% blocks/feature icon="fa-lightbulb" title="MIMIC-note" url="/iv/modules/note" %}} MIMIC-cxr provides chest x-ray data, where available, for patients from the clinical database. {{% /blocks/feature %}} {{< /blocks/section >}} {{% pageinfo color="primary" %}}
MIMIC is a publicly available database of patients admitted to the Beth Israel Deaconess Medical Center in Boston, MA, USA.
